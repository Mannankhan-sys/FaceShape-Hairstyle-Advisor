# ✨ Aesthetic AI | Professional Style Advisor

A high-end "Fashion-Tech" application that combines 3D facial landmark analysis with mathematical beauty principles. Built as a professional-grade advisor, **Aesthetic AI** provides real-time facial geometry analysis, symmetry ratings, and personalized styling curations.

---

## 💎 Premium Features

### 📐 Static Analysis Suite (Pro Grade)
Features a unified **Analysis Locking** mechanism. Once the user's face is stabilized (scanning ~25 frames), the AI "locks" all measurements—Geometry, Symmetry, and Skin Tone—providing a formal, static report that eliminates live-detection jitter.

### 🎭 Symmetry & Golden Ratio Engine
A mathematical analysis engine that evaluates facial balance:
- **Horizontal Symmetry**: Measures bilateral alignment across eyes, cheeks, and jaw.
- **Golden Ratio ($\phi$) Integration**: Evaluates facial proportions against the universal 1.618 ideal.
- **Stabilized Scoring**: Professional-grade percentage ratings with technical assessment labels.

### 🎨 Intelligent Skin Tone Analysis
Using 3-point facial sampling (Forehead and Cheeks), the AI detects:
- **Undertone Category**: Warm, Cool, or Neutral classification.
- **Precision Color Matching**: Suggests professional color palettes (e.g., "Gold & Earth", "Silver & Jewel") that best complement the user's natural complexion.

### 🔬 Cyber-Fashion Telemetry
- **Neon Glow Landmarks**: High-impact visualization of key measurement points.
- **Refined Contours**: Neon rose highlighting for major facial features.
- **Professional Mesh**: Lavender tessellation for a high-tech "Beauty Clinic" aesthetic.

### 🛠️ Engineering-Grade Dashboard
An ergonomic, **No-Scroll** UI designed for professional analysis:
- **Technical Status Badges**: Live indicators for `[ SCANNING ]` and `[ ANALYSIS LOCKED ]`.
- **Compartmentalized Metrics**: Dashboard-style grouping of all analysis data.
- **Unified Control Sidebar**: Streamlined navigation and a dedicated `🔄 Retake Analysis` workflow.

---

## 🛠️ Technical Stack
- **Python 3.10+**
- **MediaPipe:** 468-point 3D Face Mesh and Refined Landmark telemetry.
- **OpenCV:** High-performance video processing.
- **CustomTkinter:** Modern, dark-slate ergonomic interface.
- **NumPy:** Mathematical vector analysis for symmetry and color sampling.

## 📦 Quick Start

1. **Install Dependencies:**
   ```bash
   pip install customtkinter mediapipe opencv-python pillow numpy
   ```

2. **Run Application:**
   ```bash
   python gui_face_shape_modern.py
   ```

## 🖥️ Professional Workflow
1.  **Initiate Scan**: Select gender and click **Analysis Scan**.
2.  **Telemetry Sync**: Look directly at the camera while the AI scans (~2 seconds).
3.  **Result Locking**: The `[ ANALYSIS LOCKED ]` badge will confirm when measurements are finalized.
4.  **Review Curations**: Explore the "Engineered Curations" for hair and beard styles.
5.  **Re-Analyze**: Click **🔄 Retake Analysis** anytime for a fresh measurement.
